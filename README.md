# 🎯 SPAR Rewards Intelligence

**Turn customer transaction data into actionable retention strategies.**

## What It Does

This dashboard analyzes SPAR Rewards redemption data to help you understand customer behavior, identify at-risk members, and take action to retain them.

## Key Features

- **RFM Analysis** - Scores customers on Recency, Frequency & Monetary value
- **6 Customer Segments** - Champions, Loyal, Potential, One-Time, At Risk, Churned
- **Risk Scoring** - 0-100 churn prediction with priority levels
- **AI Assistant** - Answers questions about SPAR Rewards and your data
- **Action Center** - Automated marketing recommendations per customer
- **Export Ready** - Download customer lists for campaigns

## Quick Start

1. Upload your CSV (requires: Member Number, Creation Date, Redeeming Basket Value)
2. View KPIs and segment breakdown instantly
3. Filter customers by segment, spend, or priority
4. Export high-risk customer lists for retention campaigns
